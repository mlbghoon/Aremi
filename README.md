# Aremi (동선) — 일정 + 지도 캘린더 앱

날짜별로 일정을 넣으면, 그날 갈 곳들을 **가장 효율적인 순서로 묶어 지도에 동선으로** 그려주고,
**언제 출발해야 할지**까지 알려주는 모바일 웹앱. 지도 앱에서 매번 검색할 필요 없이,
미리 짜둔 일정이 곧 경로가 된다. 지나간 날은 **다이어리**로 돌아본다.

> 기획 배경·시장조사·수익화는 [`docs/`](docs/) 참고. 이 README는 **지금 실제로 동작하는 것**을 정리한다.

## 지금 되는 것

- **캘린더 (월 / 주 / 일)** — 날짜를 골라 그날 일정을 관리. 주·일은 시간대 격자, 월은 칸에 일정 표시.
- **일정(이벤트)** — 이름 + 장소(카카오 검색) + 시작~종료 시간(또는 "아무때나") + 메모 + **반복**(매일/평일/매주) + **출발 알림**.
- **동선 최적화** — "시간 고정" 약속은 순서를 지키고, "아무때나"는 그 사이를 가장 효율적으로 배치.
- **이동수단별 실제 경로** — 구간마다 🚗 자동차 / 🚇 대중교통 / 🚶 도보 선택. 각각 실제 도로·노선·인도 경로를 지도에 색으로 그린다.
- **스마트 시간표** — 실제 이동시간으로 도착/출발 시각을 계산하고, 약속에 늦으면 경고.
- **출발 알림** — 직전 일정에서 여기까지 걸리는 시간을 반영해 "지금 출발" 알림 (앱이 열려 있을 때).
- **다이어리(기록)** — "이날 뭐 했지?"를 항목 리스트 + 메모 + 지도로. (사진 첨부는 다음 단계)
- **저장** — 브라우저(localStorage)에 자동 저장.

## 화면

- **달력 화면** — 일정 관리 (추가/편집은 바텀시트 모달).
- **동선 화면** — 지도 + 하단 시트. `🗺 동선`(시간표·이동수단)과 `📖 기록`(항목·메모) 토글.
- 전체가 **폰 프레임** 안의 다크 테마 모바일 앱 형태.

## 실행

```bash
npm install
npm run dev            # 기본 http://localhost:3000
PORT=3100 npm run dev  # 다른 포트 (현재 개발 포트)
```

키가 없어도 동작한다 (지도는 간이 지도, 검색은 서울 샘플). 실제 지도·경로는 아래 키가 필요하다 →
[`docs/GET-KEYS.md`](docs/GET-KEYS.md).

## 필요한 키 (`.env.local`)

| 키                          | 용도                          | 공개/비밀              |
| --------------------------- | ----------------------------- | ---------------------- |
| `NEXT_PUBLIC_KAKAO_JS_KEY`  | 카카오 지도 + 장소 검색       | 공개 (도메인 등록으로 보호) |
| `KAKAO_REST_KEY`            | 자동차 경로 (카카오모빌리티)  | 비밀 (서버 전용)       |
| `ODSAY_KEY`                 | 대중교통 (지하철/버스, ODsay) | 비밀 (서버 전용)       |
| `TMAP_APP_KEY`              | 도보 경로 (TMap)              | 비밀 (서버 전용)       |

> ⚠️ 포트를 바꾸면 카카오 지도가 도메인 제한에 걸린다. 카카오 콘솔의 **JS SDK 도메인**에 해당 포트를
> 추가하거나 `3000`을 쓴다. (서버 API로 부르는 자동차·대중교통·도보는 포트와 무관하게 동작.)

## 기술

Next.js (App Router, TypeScript) · 카카오맵 JS SDK · 서버 API 라우트로 카카오모빌리티 / ODsay / TMap 호출
(비밀 키는 서버에만) · localStorage 저장.

## 구조

```
app/
  layout.tsx        # 폰 프레임 + 모바일 뷰포트
  page.tsx          # 달력 / 동선 / 기록 화면, 상태, 출발 알림
  globals.css       # 다크 테마
  api/directions/   # 자동차 경로 (카카오모빌리티)
  api/transit/      # 대중교통 (ODsay + loadLane 노선 geometry)
  api/walk/         # 도보 (TMap 보행자)
components/
  CalendarView.tsx  # 월/주/일 달력
  EventModal.tsx    # 일정 추가·편집 (바텀시트)
  KakaoMap.tsx      # 실제 지도 + 구간별 색 동선
  FallbackMap.tsx   # 키 없을 때 간이 지도 (SVG, 확대/이동)
  PlaceSearch.tsx   # 장소 검색 (카카오 or 샘플)
lib/
  optimize.ts       # ★ 동선 최적화 (고정/유동)
  schedule.ts       # ★ 스마트 시간표 (도착/출발/지각)
  recurrence.ts     # 반복 일정 전개
  geo.ts            # 거리/시간 계산
  date.ts           # 달력·날짜 helper
  types.ts  samplePlaces.ts  useKakao.ts
```
